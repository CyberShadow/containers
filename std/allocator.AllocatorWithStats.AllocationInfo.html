std.allocator.AllocatorWithStats.AllocationInfo
struct AllocationInfo
Per-allocation information that can be iterated upon by using . This only tracks live allocations and is useful for e.g. tracking memory leaks.

Example

       AllocatorWithStats!(Mallocator, Options.all) a;
       auto d1 = a.allocate(10);
       auto d2 = a.allocate(11);
       a.deallocate(d1);
       foreach (ref e; a.byAllocation)
       {
           writeln("Allocation module: ", e.callerModule);
       }
