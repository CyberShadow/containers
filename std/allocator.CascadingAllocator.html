std.allocator.CascadingAllocator
struct CascadingAllocator(alias make)
Given as a function that returns fresh allocators, creates an allocator that lazily creates as many allocators are needed for satisfying client allocation requests.
The management data of the allocators is stored in memory obtained from the allocators themselves, in a private linked list.
