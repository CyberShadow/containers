std.allocator.AffixAllocator
struct AffixAllocator(Allocator, Prefix, Suffix = void)
Allocator that adds some extra data before (of type ) and/or after (of type ) any allocation made with its parent allocator. This is useful for uses where additional allocation-related information is needed, such as mutexes, reference counts, or walls for debugging memory corruption errors.
If is not , must guarantee an alignment at least as large as . Suffixes are slower to get at because of alignment rounding, so prefixes should be preferred. However, small prefixes blunt the alignment so if a large alignment with a small affix is needed, suffixes should be chosen.
