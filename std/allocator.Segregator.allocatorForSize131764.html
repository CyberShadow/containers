std.allocator.Segregator.allocatorForSize
refautoauto ref allocatorForSize(size_t s)() ref auto
Composite allocators involving nested instantiations of make it difficult to access individual sub-allocators stored within. simplifies the task by supplying the allocator nested inside a that is responsible for a specific size .

Example

   alias A = Segregator!(300,
       Segregator!(200, A1, A2),
       A3);
   A a;
   static assert(typeof(a.allocatorForSize!10) == A1);
   static assert(typeof(a.allocatorForSize!250) == A2);
   static assert(typeof(a.allocatorForSize!301) == A3);
