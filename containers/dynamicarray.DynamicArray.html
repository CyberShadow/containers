containers.dynamicarray.DynamicArray
public struct DynamicArray(T, bool supportGC = true)
Array that is able to grow itself when items are appended to it. Uses reference counting to manage memory and malloc/free/realloc for managing its storage.

Parameters

Tthe array element type
supportGCtrue if the container should support holding references to GC-allocated memory.
