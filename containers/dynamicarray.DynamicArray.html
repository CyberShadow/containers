containers.dynamicarray.DynamicArray
public struct DynamicArray(T, Allocator = Mallocator, bool supportGC = shouldAddGCRange!T)
Array that is able to grow itself when items are appended to it. Uses malloc/free/realloc to manage its storage.

Parameters

Tthe array element type
Allocatorthe allocator to use. Defaults to Mallocator.
supportGCtrue if the container should support holding references to GC-allocated memory.
