containers.unrolledlist.UnrolledList
public struct UnrolledList(T, Allocator = Mallocator, bool supportGC = shouldAddGCRange!T, size_t cacheLineSize = 64)
Unrolled Linked List.
Nodes are (by default) sized to fit within a 64-byte cache line. The number of items stored per node can be read from the nodeCapacity field.

See Also

http://en.wikipedia.org/wiki/Unrolled_linked_list
