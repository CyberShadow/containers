containers.openhashset.OpenHashSet
public struct OpenHashSet(T, Allocator = Mallocator, alias hashFunction = generateHash!T, bool supportGC = shouldAddGCRange!T)
Simple open-addressed hash set. Use this instead of HashSet when the size and quantity of the data to be inserted is small.

Parameters

Tthe element type of the hash set
Allocatorthe allocator to use. Defaults to Mallocator.
hashFunctionthe hash function to use
supportGCif true, calls to GC.addRange and GC.removeRange will be used to ensure that the GC does not accidentally free memory owned by this container.
